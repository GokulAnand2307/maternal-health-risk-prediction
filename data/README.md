# Dataset setup

Download the public **Maternal Health Risk Data Set** from the UCI Machine Learning Repository and save the CSV as:

```text
data/maternal_health_risk.csv
```

Expected columns:

| Column | Meaning |
|---|---|
| `Age` | Age in years |
| `SystolicBP` | Systolic blood pressure in mmHg |
| `DiastolicBP` | Diastolic blood pressure in mmHg |
| `BS` | Blood sugar in mmol/L |
| `BodyTemp` | Body temperature |
| `HeartRate` | Heart rate in beats per minute |
| `RiskLevel` | `low risk`, `mid risk` or `high risk` |

The raw dataset is not duplicated in this repository so that its original source, licence and citation remain clear. Do not substitute patient-identifiable or confidential clinical records.
